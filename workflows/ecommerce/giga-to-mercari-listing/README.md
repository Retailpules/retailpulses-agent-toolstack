# GigaB2B to Mercari Listing

Convert selected GigaB2B product data into a Mercari-ready listing draft with validated Japanese copy, basic QA, and review artifacts.

## Status

`assisted` — agents may help execute steps after approval, but this workflow must not create live Mercari listings yet.

## Output

- Mercari listing draft JSON
- QA report
- Missing data report
- Operator review checklist

## Not allowed in MVP

- Direct live Mercari upload
- Price changes without approval
- Customer messages
- Production deployment
